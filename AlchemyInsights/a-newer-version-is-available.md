# <a name="delete-the-office-document-cache"></a>Löschen des Office-Dokumentcaches

Wenn Sie eine Excel Eingabeaufforderung erhalten haben, dass **eine neuere Version der Datei verfügbar ist,** versuchen Sie, den Office Dokumentcache zu löschen, indem Sie die Anweisungen unter ["Löschen Ihres Office Dokumentcaches"](https://support.office.com/article/b1d3765e-d71b-4bb8-99ca-acd22c42995d)befolgen. Versuchen Sie dann, die Datei erneut zu öffnen.
