# <a name="about-threat-analytics"></a>Informationen zu Bedrohungsanalysen

Die Bedrohungsanalyse ist die Produktlösung für die Bedrohungserkennung von Microsoft, die von experten Microsoft-Sicherheitsexperten stammt. Es wurde entwickelt, um Sicherheitsteams dabei zu unterstützen, sich neuen Bedrohungen so effizient wie möglich zu stellen, einschließlich aktiver Bedrohungsakteure und ihrer Kampagnen, beliebter und neuer Angriffstechniken, kritischer Sicherheitsrisiken, allgemeiner Angriffsflächen und weit verbreiteter Schadsoftware. Im Rahmen der einheitlichen Sicherheitserfahrung steht Bedrohungsanalysen Microsoft 365 Defender Abonnenten und Microsoft Defender für Endpunkt und Microsoft Defender für Office E5-Lizenzinhabern zur Verfügung. 

Weitere Informationen finden Sie unter [Nachverfolgen und Reagieren auf neue Bedrohungen](https://docs.microsoft.com/microsoft-365/security/defender/threat-analytics).

Wenn Sie ein Microsoft 365 Defender Benutzer sind, unabhängig davon, ob Sie das Produkt auswerten oder über eine Volllizenz verfügen, können Sie jederzeit auf Bedrohungsanalysen und Bedrohungsberichte zugreifen.
