# <a name="adding-external-users"></a>Hinzufügen externer Benutzer

Die Schritte zum Hinzufügen von externen/Gastbenutzern können je nachdem, was Sie benötigen, um darauf zugreifen zu können, unterschiedlich sein. Wählen Sie eine der folgenden Optionen aus:
  
- [Übersicht über die externe Freigabe in SharePoint Online](https://docs.microsoft.com/sharepoint/external-sharing-overview)

- [Gastzugriff in Microsoft 365-Gruppen](https://support.microsoft.com/office/adding-guests-to-microsoft-365-groups-bfc7a840-868f-4fd6-a390-f347bf51aff6)

- [Gastzugriff in Microsoft Teams](https://docs.microsoft.com/microsoftteams/guest-access-checklist)
