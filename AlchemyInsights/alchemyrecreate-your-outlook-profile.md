# <a name="create-an-outlook-profile"></a>Erstellen eines Outlook Profils

Um Ihr Outlook Profil zu erstellen oder neu zu erstellen, können Sie das [Outlook Profiltool](https://aka.ms/SaRA-OutlookSetupProfile-Alchemy) auf dem betroffenen Computer ausführen.

Das Tool führt verschiedene Überprüfungen durch, und wenn Sie bereit sind, erstellt es ein Outlook Profil mit Ihrer E-Mail-Adresse.
