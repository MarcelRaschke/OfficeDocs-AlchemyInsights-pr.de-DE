# <a name="automatically-record-a-teams-meeting"></a>Automatisches Aufzeichnen einer Teams-Besprechung

Um eine Teams-Besprechung automatisch aufzuzeichnen, muss der Besprechungsorganisator die **Besprechungsoptionen** bearbeiten, bevor die Besprechung beginnt, indem er " **Automatisch aufzeichnen"** auf **"Ja**" festlegt.

Weitere Informationen zu Besprechungsoptionen finden Sie unter [Ändern der Teilnehmereinstellungen für eine Teams-Besprechung](https://support.microsoft.com/office/change-participant-settings-for-a-teams-meeting-53261366-dbd5-45f9-aae9-a70e6354f88e).
