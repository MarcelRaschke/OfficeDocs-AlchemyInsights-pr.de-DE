# <a name="change-the-topic-center-url"></a>Ändern der Themencenter-URL

Als Microsoft 365-Themenadministrator können Sie die Themencenter-URL ändern oder in einigen Fällen ganz mit dem Themencenter beginnen. In beiden Fällen müssen Sie ein Supportticket öffnen.
