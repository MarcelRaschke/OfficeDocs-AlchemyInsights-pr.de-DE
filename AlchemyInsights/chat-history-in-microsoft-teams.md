# <a name="viewing-chat-history-in-microsoft-teams"></a>Anzeigen des Chatverlaufs in Microsoft Teams

Um Teams Chatverlauf anzuzeigen, können Sie die [Inhaltssuche](https://sip.protection.office.com/contentsearchbeta?ContentOnly=1) im [Security & Compliance Center](https://sip.protection.office.com/insightdashboard)verwenden, in dem Sie das Postfach durchsuchen können, das einer Microsoft 365-Gruppe oder einem Microsoft-Team zugeordnet ist.

Weitere Informationen zu den Suchfunktionen im Security and Compliance Center finden Sie unter Suchen Microsoft Teams [und Microsoft 365-Gruppen.](https://docs.microsoft.com/microsoft-365/compliance/content-search)
