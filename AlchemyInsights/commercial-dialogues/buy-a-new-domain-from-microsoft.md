# <a name="buy-a-new-domain-from-microsoft"></a>Kaufen einer neuen Domäne von Microsoft

Führen Sie die folgenden Schritte aus, um eine neue Domäne von Microsoft zu kaufen:

1. Navigieren Sie zu **Setup > Domänen** , und klicken Sie auf **"Domäne kaufen**". 
2. Befolgen Sie die Anweisungen des Assistenten.
