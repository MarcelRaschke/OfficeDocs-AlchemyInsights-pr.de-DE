# <a name="i-dont-see-automatic-replies"></a>Ich sehe keine automatischen Antworten

Wenn nach dem Auswählen der **Datei** keine automatischen Antworten angezeigt werden, verwenden Sie wahrscheinlich ein Gmail-, Yahoo- oder anderes POP- oder IMAP-Konto, das das Feature Outlook automatische Antworten nicht unterstützt. Sie können eine Regel einrichten, die auf eingehende Nachrichten antwortet, jedoch nur, wenn Sie Outlook ausführen lassen.
