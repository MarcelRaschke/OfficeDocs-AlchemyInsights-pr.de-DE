# <a name="install-deploy-and-configure-mdatp-on-a-mac"></a>Installieren, Bereitstellen und Konfigurieren von MDATP auf einem Mac

Sie können MDATP [manuell](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/mac-install-manually) oder mithilfe eines Verwaltungstools bereitstellen:

- [Microsoft Intune](https://go.microsoft.com/fwlink/?linkid=2144548)
- [Jamf](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/mac-install-with-jamf)
- [Verwaltung von Mobilgeräten (Mobile Device Management, MDM)](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/mac-install-with-other-mdm)

Weitere Informationen finden Sie unter [Installieren von MDATP für Mac](https://go.microsoft.com/fwlink/?linkid=2144672)
