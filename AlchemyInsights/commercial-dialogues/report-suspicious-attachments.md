# <a name="report-suspicious-attachments"></a>Melden verdächtiger Anlagen

1. Wählen Sie im Flyout **"Neue Übermittlung"** die Option **"Anlage" aus,** und laden Sie dann die Datei hoch, die Sie scannen möchten.
    > [!NOTE]
    > Derzeit gilt eine Beschränkung von 3 MB. Wenn die Datei 3 MB überschreitet, sollten Sie sie stattdessen mit einer E-Mail-ID übermitteln.
2. Wenn Sie **"Sollte blockiert" ausgewählt haben,** wird das Element automatisch als **Schadsoftware** gekennzeichnet.
3. Wählen Sie **Senden** aus.
