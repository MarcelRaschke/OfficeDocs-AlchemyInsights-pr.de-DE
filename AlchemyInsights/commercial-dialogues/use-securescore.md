# <a name="use-securescore"></a>Verwenden von SecureScore

Secure Score ist ein Sicherheitsanalysetool, das empfiehlt, was Sie tun können, um risiken weiter zu reduzieren. Secure Score überprüft Ihre Einstellungen und Aktivitäten in Office 365 und vergleicht diese mit einer von Microsoft aufgestellten Basislinie. Anschließend erhalten Sie eine Bewertung basierend auf Ihrer Anwendung der bewährten Methoden für die Wahrung der Sicherheit.

Möchten Sie die Sicherheitsbewertung ausprobieren? Wenn Sie bei Office 365 als globaler Administrator angemeldet sind, wechseln Sie zu [SecureScore.com.](https://docs.microsoft.com/microsoft-365/security/defender/microsoft-secure-score/)
