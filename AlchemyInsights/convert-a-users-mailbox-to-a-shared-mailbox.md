# <a name="convert-a-user-mailbox-to-a-shared-mailbox"></a>Konvertieren eines Benutzerpostfachs in ein freigegebenes Postfach

Wenn Sie das Postfach eines einzelnen Benutzers in ein freigegebenes Postfach konvertieren, werden alle vorhandenen E-Mail- und Kalenderelemente in einem freigegebenen Postfach aufbewahrt, auf das mehrere Personen zugreifen können. Sie können ein freigegebenes Postfach auch wieder in ein privates Benutzerpostfach konvertieren.
  
Weitere Informationen finden Sie unter [Konvertieren eines Benutzerpostfachs in ein freigegebenes Postfach.](https://docs.microsoft.com/microsoft-365/admin/email/convert-user-mailbox-to-shared-mailbox)
