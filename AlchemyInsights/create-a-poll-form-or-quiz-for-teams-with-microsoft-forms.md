# <a name="create-a-poll-form-or-quiz-for-teams-with-microsoft-forms"></a>Erstellen einer Umfrage, eines Formulars oder einer Prüfung für Teams mit Microsoft Forms

Möchten Sie ein Formular, eine Umfrage, eine Umfrage oder ein Quiz in Microsoft Teams erstellen? Microsoft Forms kann Ihnen helfen!

 - [Forms aktivieren oder deaktivieren](https://docs.microsoft.com/microsoft-forms/turn-off-turn-on-microsoft-forms), für Ihre Organisation oder nur für bestimmte Benutzer aktivieren.
 
 - [Erstellen, Bearbeiten und Zusammenarbeit](https://support.microsoft.com/office/work-with-colleagues-to-create-edit-and-review-forms-in-microsoft-teams-333b97a3-41d9-48bc-a1cb-84a96bd44e14) an Formularen in Microsoft Teams.
 
 - [Häufig gestellte Fragen](https://support.microsoft.com/office/forms-basics-1dd58027-40dc-42d0-9ca4-80ddecc5c696) zu Microsoft Forms.
