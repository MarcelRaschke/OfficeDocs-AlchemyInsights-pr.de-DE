# <a name="data-location"></a>Datenspeicherort

Sie können den Standort Ihres Mandanten im Admin Center anzeigen, indem Sie zur [**Profilseite "Organisation**](https://admin.microsoft.com/AdminPortal/Home#/Settings/OrganizationProfile)" navigieren und "**Datenspeicherort**" auswählen.

Für Anwendungen, die Sie nicht abonniert haben, lesen Sie, [wo Ihre Microsoft 365 Kundendaten gespeichert werden](https://docs.microsoft.com/office365/enterprise/o365-data-locations).
