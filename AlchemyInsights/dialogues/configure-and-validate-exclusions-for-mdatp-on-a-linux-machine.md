# <a name="configure-and-validate-exclusions-for-mdatp-on-a-linux-machine"></a>Konfigurieren und Überprüfen von Ausschlüssen für MDATP auf einem Linux-Computer

Sie können bestimmte Dateien, Ordner, Prozesse und prozessge öffnende Dateien aus MDATP-Scans ausschließen. Ausschlüsse verhindern eine falsche Erkennung von Software und Dateien, die eindeutig oder an Ihre Organisation angepasst sind. Ausschlüsse helfen auch bei der Verringerung von Leistungsproblemen, die durch MDATP verursacht werden.

Weitere Informationen finden Sie unter [Configure and validate exclusions for MDATP for Linux](https://go.microsoft.com/fwlink/?linkid=2144517).

> [!IMPORTANT]
> Die in diesem Artikel beschriebenen Ausschlüsse gelten nicht für andere Funktionen von MDATP für Linux, einschließlich Endpunkterkennung und -reaktion (EDR). Dateien, die Sie mithilfe der in diesem Artikel beschriebenen Methoden ausschließen, können weiterhin EDR-Warnungen und andere Erkennungsfunktionen auslösen.
