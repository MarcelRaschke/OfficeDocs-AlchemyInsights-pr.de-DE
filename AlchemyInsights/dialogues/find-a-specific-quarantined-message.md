# <a name="find-a-specific-quarantined-message"></a>Suchen einer bestimmten Nachricht in Quarantäne

1. Wechseln Sie zum [Security & Compliance Center](https://go.microsoft.com/fwlink/p/?linkid=2077143).
2. Wechseln Sie zu **Bedrohungsverwaltung**  >  **Überprüfen der**  >  **Quarantäne**.
3. Wählen Sie **Suchen** aus.
4. Wählen Sie eine beliebige Kombination von Bedingungen aus, indem Sie die Kontrollkästchen neben ihnen aktivieren (Sie können derzeit keine Platzhalter verwenden).
