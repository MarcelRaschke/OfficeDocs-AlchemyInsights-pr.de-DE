# <a name="distribution-group-not-showing-in-groups-list"></a>Verteilergruppe wird in Gruppenliste nicht angezeigt

Es kann bis zu 60 Minuten dauern, bis Ihre neue Verteilerliste im Exchange Admin Center Center angezeigt wird.
  
Wenn Sie jetzt Mitglieder hinzufügen oder andere Änderungen vornehmen möchten, verwenden Sie bitte das Feature "Gruppen" im [Exchange Admin Center](https://outlook.office365.com/ecp/?rfr=Admin_o365&amp;exsvurl=1&amp;mkt=en-US.aspx).
