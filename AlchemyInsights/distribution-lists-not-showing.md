# <a name="distribution-group-not-showing-in-groups-list"></a>Verteilergruppe wird nicht in der Gruppenliste angezeigt

Es kann bis zu 60 Minuten dauern, bis Ihre neue Verteilerliste im Exchange Admin Center angezeigt wird.
  
Wenn Sie gerade Mitglieder hinzufügen oder andere Bearbeitungen erstellen möchten, verwenden Sie bitte das Feature Gruppen im [Exchange Admin Center](https://outlook.office365.com/ecp/?rfr=Admin_o365&amp;exsvurl=1&amp;mkt=en-US.aspx).
