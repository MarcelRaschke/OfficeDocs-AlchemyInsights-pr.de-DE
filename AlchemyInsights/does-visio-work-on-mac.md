# <a name="does-visio-work-on-a-mac"></a>Funktioniert Visio auf einem Mac?

Es gibt keine Visio-Desktopanwendung für Mac. Solange dem Benutzer, der die App benötigt, jedoch eine [Visio Onlinelizenz zugewiesen](https://docs.microsoft.com/microsoft-365/admin/add-users/add-users) ist, kann er [Diagramme in Visio Online in einem Webbrowser anzeigen, erstellen und bearbeiten](https://support.office.com/article/06f04845-91b8-4e8f-881f-a43c970735fc?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1799).
