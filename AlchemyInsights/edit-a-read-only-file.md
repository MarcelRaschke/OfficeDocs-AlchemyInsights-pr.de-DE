# <a name="edit-a-read-only-file"></a>Bearbeiten einer schreibgeschützten Datei

Wenn Sie in Dokumentbibliotheken auf den Namen eines Dokuments klicken, wird es möglicherweise unerwartet vom Office-Programm als schreibgeschützte Datei geöffnet. Klicken Sie zum Bearbeiten auf **"Öffnen** " (die drei Punkte), und klicken Sie dann auf **"Bearbeiten".**
