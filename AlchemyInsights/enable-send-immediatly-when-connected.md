# <a name="enable-send-immediately-when-connected"></a>Sofort senden aktivieren, wenn eine Verbindung besteht
 
1. Klicken Sie auf der Registerkarte Datei auf **Optionen**.

2. Klicken Sie im Dialogfeld Outlook-Optionen auf **Erweitert**.

3. Klicken Sie im Abschnitt Senden und Empfangen auf, um Sofort **senden zu aktivieren, wenn eine Verbindung besteht.** Klicken Sie auf **OK**.
