# <a name="find-lost-or-missing-files-in-onedrive-or-sharepoint"></a>Suchen verlorener oder fehlender Dateien in OneDrive oder SharePoint

- Überprüfen Sie den Papierkorb, indem Sie zu Ihrem OneDrive oder zur SharePoint-Website im Web navigieren und im linken Bereich auf den Papierkorb klicken. (Wenn der Papierkorb im linken Bereich nicht angezeigt wird, klicken Sie auf das Symbol "Einstellungen", und wählen Sie dann **"Websiteinhalte" aus**. **Der Papierkorb** befindet sich ganz rechts auf der Befehlsleiste oben.) 
    
- Wenn Sie die Dateien auf Ihrem Computer synchronisieren, überprüfen Sie den Papierkorb auf Ihrem Computer. 
    
- Wenn das gesuchte Element nicht angezeigt wird und es kürzlich gelöscht wurde, kann es möglicherweise von einem Websitesammlungsadministrator aus dem Papierkorb der Websitesammlung wiederhergestellt werden. Wenn Sie der Websitesammlungsadministrator sind, lesen Sie " [Wiederherstellen gelöschter Elemente aus dem Papierkorb der Websitesammlung](https://support.microsoft.com/office/restore-items-in-the-recycle-bin-that-were-deleted-from-sharepoint-or-teams-6df466b6-55f2-4898-8d6e-c0dff851a0be)".
    
[Weitere Hilfe beim Wiederherstellen gelöschter Dateien und Ordner in OneDrive](https://go.microsoft.com/fwlink/?linkid=872872)
