# <a name="get-in-app-support-for-the-outlook-mobile-app"></a>In-App-Unterstützung für die mobile Outlook-App erhalten

1. Öffnen Sie in der mobilen Outlook-App das **Menü**.

2. Wählen Sie das Fragezeichen aus, um den Bildschirm **&amp; Hilfefeedback zu** öffnen.

3. Tippen **Sie auf Support kontaktieren,** und erläutern Sie Ihr Problem.

4. Tippen **Sie auf Senden,** und Sie erhalten eine In-App-Benachrichtigung, wenn sie von einem Support-Agent kontaktiert werden.

Ausführliche Anweisungen finden Sie unter [Get in-app help for Outlook for iOS and Android](https://support.office.com/article/218a22d1-9fa5-4889-b689-de1c63493243.aspx#ID0EAABAAA=Contact_Support).
