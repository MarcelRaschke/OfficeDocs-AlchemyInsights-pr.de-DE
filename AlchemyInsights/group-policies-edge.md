# <a name="use-update-related-group-policies-available-in-microsoft-edge"></a>Verwenden von updatebezogenen Gruppenrichtlinien, die in Microsoft Edge

Verwenden Sie die in Microsoft Edge (Version 77 oder höher) verfügbaren Richtlinien, um zu steuern, wie und wann Microsoft Edge aktualisiert wird. Weitere Informationen finden Sie unter [Microsoft Edge – Richtlinien aktualisieren.](/DeployEdge/microsoft-edge-update-policies#available-policies)
