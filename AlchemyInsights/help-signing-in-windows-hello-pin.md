# <a name="help-signing-in-with-windows-hello-pin"></a>Hilfe bei der Anmeldung mit Windows Hello-PIN

- Wenn Sie die PIN ändern, zurücksetzen oder wiederherstellen möchten (wenn Sie angemeldet sind), besuchen Sie[„PIN ändern“, wenn Sie bereits bei Ihrem Gerät angemeldet sind](https://support.microsoft.com/windows/change-your-pin-when-you-re-already-signed-in-to-your-device-0bd2ab85-b0df-c775-7aef-1324f2114b19).

- Informationen zum Ändern, Zurücksetzen oder Wiederherstellen Ihrer PIN (wenn Sie nicht angemeldet sind) finden Sie unter [Zurücksetzen der PIN, wenn Sie nicht bei Windows 10 angemeldet sind](https://support.microsoft.com/windows/reset-your-pin-when-you-aren-t-signed-in-to-windows-10-a386c519-3ab2-b873-1e9b-bb228a98b904).

- Um die Windows Hello-PIN einzurichten, besuchen Sie [Erfahren Sie mehr über Windows Hello und richten Sie es ein](https://support.microsoft.com/windows/learn-about-windows-hello-and-set-it-up-dae28983-8242-bb2a-d3d1-87c9d265a5f0).
