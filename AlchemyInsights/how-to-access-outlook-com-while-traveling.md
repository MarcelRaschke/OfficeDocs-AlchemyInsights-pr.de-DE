# <a name="how-to-access-outlookcom-while-traveling"></a>So greifen Sie auf Reisen auf Outlook.com zu

Um während der Reise auf Outlook.com zuzugreifen, stellen Sie sicher, dass Die Sicherheitsinformationen Ihres Microsoft-Kontos – eine alternative E-Mail-Adresse oder Telefonnummer, unter der Sie Sicherheitscodes erhalten können – auf dem neuesten Stand sind. Weitere Informationen finden Sie unter [Zugreifen auf Outlook.com auf Reisen](https://support.office.com/article/c44f16da-7156-4890-853c-286aafeda87e?wt.mc_id=Office_Outlook_com_Alchemy).
