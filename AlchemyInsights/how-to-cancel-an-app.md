# <a name="how-to-cancel-software-as-a-service-apps"></a>Abbrechen von Software-as-a-Service-Apps

1. Im Admin Center zur Seite **Abrechnung**  >  [Ihre Produkte.](https://go.microsoft.com/fwlink/p/?linkid=842054)
2. Scrollen Sie **auf** der Seite Produkte nach unten zum Abschnitt **Apps,** und suchen Sie nach dem Abonnement, das Sie kündigen möchten. 
3. Wählen Sie die drei Punkte (weitere Aktionen) aus, und wählen Sie dann **Abonnement abbrechen aus.**
4. Wählen Sie im Bereich **Abonnement kündigen** einen Grund aus, warum Sie kündigen. Geben Sie optional Feedback.
5. Wählen Sie **Speichern** aus.
