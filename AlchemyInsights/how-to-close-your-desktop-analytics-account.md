# <a name="how-to-close-your-desktop-analytics-account"></a>So schließen Sie Ihr Desktop Analytics-Konto

Wenn Sie Desktop Analytics in Ihrer Umgebung einrichten und dann entscheiden, dass Sie es entfernen müssen, verwenden Sie den Prozess im folgenden Link:

[So schließen Sie Ihr Konto](https://docs.microsoft.com/configmgr/desktop-analytics/account-close)

Der erste Schritt besteht darin, Microsoft-Support zu kontaktieren. Öffnen Sie einen Supportfall, um Ihr Desktop Analytics Konto zu schließen. Fahren Sie erst mit weiteren Schritten fort, nachdem Sie die Bestätigung erhalten haben, dass Microsoft Ihr Konto geschlossen hat.
