# <a name="how-to-install-office-on-an-ipad"></a>Installieren von Office auf einem iPad

- Lassen Sie den Benutzer [auf den Downloadlink](https://support.office.com/article/9df6d10c-7281-4671-8666-6ca8e339b628?wt.mc_id=Alchemy_ClientDIA) von ihrem mobilen Gerät für Word, Excel, PowerPoint und OneNote tippen.

- Lassen Sie den Benutzer die App starten, tippen **Sie auf "Anmelden"** , und geben Sie seine Geschäfts-, Schul- oder Unianmeldeinformationen ein.
