# <a name="how-to-move-a-onenote-notebook"></a>So verschieben Sie ein OneNote Notizbuch

* Um ein OneNote Notizbuch in OneDrive zu verschieben, wechseln Sie zu **"****FileShareMove** >  >  **Notebooks**" in OneNote 2016. Weitere Informationen finden Sie unter [Verschieben eines OneNote Notizbuchs in OneDrive](https://support.office.com/article/Move-a-OneNote-notebook-to-OneDrive-0af0a141-0bdf-49ab-9e50-45dbcca44082).

* Weitere Informationen zu bewährten Methoden bei der Vorbereitung auf das Verschieben von Notizbüchern finden Sie [unter OneNote bewährten Methoden für die Synchronisierung](https://support.microsoft.com/help/2819334/onenote-syncing-best-practices).

* [Das Verschieben eines OneNote für Windows Notizbuch, das Sie für andere freigegeben haben,](https://support.office.com/article/Move-a-OneNote-for-Windows-notebook-that-you-ve-shared-with-others-56c7659e-1850-49a6-8874-e2db6b440cd4) bietet weitere Informationen zum Verschieben freigegebener Notizbücher.
