# <a name="use-javascript-in-webview2"></a>Verwenden von JavaScript in WebView2

MitHilfe von JavaScript in WebView2-Steuerelementen können Sie systemeigene Apps an Ihre Anforderungen anpassen, grundlegende WebView2-Funktionen skizzieren und Szenarien hinzufügen, um das Programm besser zu verstehen. Weitere Informationen finden Sie unter [Verwenden von JavaScript in WebView für erweiterte Szenarien.](/microsoft-edge/webview2/how-to/javascript)
