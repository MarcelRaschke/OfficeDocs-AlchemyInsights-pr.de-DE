# <a name="learn-about-microsoft-365-groups"></a>Informationen zu Microsoft 365-Gruppen

Microsoft 365 Gruppen ist ein Dienst, der mit den bereits verwendeten Office 365 Tools arbeitet, sodass Sie mit Ihren Teamkameraden zusammenarbeiten können, wenn Sie Dokumente schreiben, Tabellenkalkulationen erstellen, an Projektplänen arbeiten, Besprechungen planen oder E-Mails senden. Mehr dazu erfahren Sie unter [Weitere Informationen zu Microsoft 365 Groups](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2).
