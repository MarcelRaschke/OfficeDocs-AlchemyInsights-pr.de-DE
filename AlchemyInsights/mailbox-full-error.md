# <a name="ndr-with-error-code-554-522-mailbox-full"></a>NDR mit Fehlercode 554 5.2.2 Postfach voll

Wenn Sie NDR mit dem Fehlercode "554 5.2.2 Postfach voll" erhalten, wenn Sie E-Mails an einen E-Mail-aktivierten öffentlichen Ordner senden:  

Verwenden Sie die Schritte in [diesem](https://aka.ms/554522) Artikel, um sicherzustellen, dass das Postfachkontingent für öffentliche Ordner oder öffentliche Ordner nicht erreicht wurde.
