# <a name="microsoft-bookings-payment-options"></a>Microsoft Bookings – Zahlungsoptionen

Die Zahlungsoptionen von Microsoft Bookings wurden im März 2020 eingestellt. Organisationen verwenden jetzt Lösungen von Drittanbietern, um Zahlungsoptionen zu Bookings hinzuzufügen.
