# <a name="microsoft-defender-smartscreen"></a>Microsoft Defender Smartscreen

Wenn Sie den neuen Microsoft Edge verwenden, hilft Ihnen Microsoft Defender SmartScreen dabei, gemeldete Phishing- und Malware-Websites zu identifizieren und fundierte Entscheidungen über Downloads zu treffen.

Weitere Informationen finden Sie unter [Was ist SmartScreen und wie kann es mich schützen?](https://support.microsoft.com/microsoft-edge/what-is-smartscreen-and-how-can-it-help-protect-me-1c9a874a-6826-be5e-45b1-67fa445a74c8).
