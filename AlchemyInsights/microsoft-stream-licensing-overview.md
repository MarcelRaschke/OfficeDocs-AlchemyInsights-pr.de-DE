# <a name="microsoft-stream-licensing-overview"></a>Übersicht über die Microsoft Stream-Lizenzierung

Microsoft Stream ist ein Clouddienst, der pro Benutzerabonnement lizenziert ist. Sie ist als Teil berechtigter Microsoft 365-Abonnements verfügbar. Informationen zu jedem Abonnementtyp finden Sie unter Microsoft 365-Preise.

- [Übersicht über die Microsoft Stream-Lizenzierung](https://docs.microsoft.com/stream/license-overview)
