# <a name="stalleddueto-errors-during-migration"></a>Fehler "AngehaltenedDueTo" während der Migration

Möglicherweise werden verschiedene Fehler angezeigt, die mit "StilledDueTo..." beginnen. während der Migration zu Exchange Online. Viele dieser Fehler sind völlig normal. Weitere Informationen finden Sie unter [ressourcenbasierte Drosselung und Priorisierung in Exchange Online Migrationen](https://techcommunity.microsoft.com/t5/exchange-team-blog/resource-based-throttling-and-prioritization-in-exchange-online/ba-p/608020).
