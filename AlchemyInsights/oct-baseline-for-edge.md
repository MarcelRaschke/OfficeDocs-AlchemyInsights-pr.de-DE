# <a name="view-the-october-2019-baseline-for-microsoft-edge-versions-77-and-later"></a>Anzeigen des Basisplans vom Oktober 2019 für Microsoft Edge, Version 77 und höher

Microsoft Intune unterstützt die Basisplaneinstellungen für Microsoft Edge. Die Basisstandardwerte stellen die empfohlene Konfiguration für den Browser dar. sie stimmen möglicherweise nicht mit den Standardwerten für andere Sicherheitsbaselines überein.

Weitere Informationen finden Sie unter [Microsoft Edge-Basisplan für Oktober 2019 (Edge-Versionen 77 und höher)](https://docs.microsoft.com/mem/intune/protect/security-baseline-settings-edge?pivots=edge-october-2019).
