# <a name="microsoft-365-apps-missing"></a>Microsoft 365 fehlende Apps

Wenn Sie Office Anwendungen auf Ihrem Computer nach einer Installation oder einem Update nicht finden können, finden Sie [Office Anwendungen nicht in Windows 10, Windows 8 oder Windows 7?](https://support.office.com/article/Can-t-find-Office-applications-in-Windows-10-Windows-8-or-Windows-7-907ce545-6ae8-459b-8d9d-de6764a635d6). Wenn Sie die Office Verknüpfungen im Startmenü oder unter **"Alle Apps**" nicht finden können, finden Sie unter [Microsoft 365 apps are missing from all apps on Windows 10](https://support.office.com/article/office-apps-are-missing-from-all-apps-on-windows-10-5bc123f6-655d-4736-ad61-b0b9d1cde5bc). 

Versuchen Sie[, eine Office Anwendung zu reparieren](https://support.office.com/article/repair-an-office-application-7821d4b6-7c1d-4205-aa0e-a6b40c5bb88b), wenn Office Verknüpfungen nach der Anmeldung oder dem Neustart des Computers nicht mehr angezeigt wurden. 

**Hinweis:** If Microsoft 365 apps are missing for installation on the Office portal, go to [Troubleshooting Office](https://support.office.com/article/troubleshoot-installing-office-35ff2def-e0b2-4dac-9784-4cf212c1f6c2) and go to **Office for businessCommon** >  **issuesYou** >  **don't see an option to install Office**.
