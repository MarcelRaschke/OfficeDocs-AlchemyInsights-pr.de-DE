# <a name="onedrive-puid-mismatch"></a>OneDrive PUID-Konflikt

Es sieht so aus, als ob in OneDrive for Business ein [ID-Konflikt](https://docs.microsoft.com/sharepoint/troubleshoot/administration/access-denied-or-need-permission-error-sharepoint-online-or-onedrive-for-business#when-accessing-a-onedrive-site) vorkommt. Führen Sie die folgenden Schritte aus, damit wir Sie frühestens unterstützen können:

1. Klicken Sie auf **"Support kontaktieren"** und dann auf **"Neue Serviceanfrage".**
2. Belassen Sie den Titel und die Beschreibung als **OneDrive Site User ID Mismatch**.
3. Nachdem das Ticket übermittelt wurde, aktualisieren Sie es mit den folgenden Informationen:
    - OneDrive website-URL, die betroffen ist.
    - Benutzerprinzipalname (UPN) des betroffenen Benutzers.
