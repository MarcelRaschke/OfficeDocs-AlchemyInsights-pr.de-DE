# <a name="read-receipts"></a>Lesebestätigungen

Sie können keinen Lesebeleg in Outlook.com anfordern, aber Sie können Lesebestätigungen für Outlook.com-Nachrichten anfordern, die mit Outlook für Windows gesendet werden. Weitere Informationen finden Sie unter [Hinzufügen und Anfordern von Lesebestätigungen und Übermittlungsbenachrichtigungen](https://support.office.com/article/a34bf70a-4c2c-4461-b2a1-12e4a7a92141?wt.mc_id=Office_Outlook_com_Alchemy).
  
Wenn Sie auswählen möchten, wie Outlook.com auf Lesebestätigungsanforderungen antwortet, wechseln Sie zu "[Lesebestätigungseinstellungen"](https://outlook.live.com/mail/options/mail/handling/readReceipts).
