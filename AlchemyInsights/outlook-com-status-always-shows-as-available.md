# <a name="my-status-always-shows-as-available"></a>Mein Status wird immer als verfügbar angezeigt

Jedes Mal, wenn Sie sich bei Outlook.com anmelden, wird Ihr Kontostatus auf "Verfügbar" zurückgesetzt. Sie können verhindern, dass sich Ihr Status ändert, indem Sie sich nicht von Outlook.com abmelden. Um beim Schließen des Browsers angemeldet zu bleiben, aktivieren Sie das Feld "**Angemeldet bleiben**" auf der Anmeldeseite Outlook.com.
