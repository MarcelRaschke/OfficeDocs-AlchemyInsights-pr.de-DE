# <a name="cant-download-files-in-chrome"></a>Dateien können in Chrome nicht heruntergeladen werden

**Problem** Wir sind uns eines Problems bewusst, das verhindert, dass einige Kunden Dateien herunterladen können, wenn sie den Google Chrome-Browser verwenden. 

**Problemumgehung** Verwenden Sie [Microsoft Edge](https://www.microsoft.com/windows/microsoft-edge) oder einen anderen Webbrowser.
Erfahren Sie mehr über das [Herunterladen von Dateien und Outlook.com-Anlagen](https://support.office.com/article/8d7c1ea7-4e5f-44ce-bb6e-c5fcc92ba9ab?wt.mc_id=Office_Outlook_com_Alchemy).
