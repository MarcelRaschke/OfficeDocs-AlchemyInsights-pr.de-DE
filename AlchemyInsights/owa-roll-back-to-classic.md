# <a name="the-new-outlook-on-the-web"></a>Die neue Outlook im Web

Es ist nicht mehr möglich, zur klassischen Outlook im Web zurückzukehren. Wenn Sie Feedback zu der neuen Benutzeroberfläche geben möchten, wechseln Sie zum [UserVoice-Forum.](https://go.microsoft.com/fwlink/?linkid=2103182) Sie können über die Ideen oder Probleme abstimmen, die Sie adressieren möchten, und wir reagieren direkt auf die wichtigsten Kommentare und Vorschläge.
