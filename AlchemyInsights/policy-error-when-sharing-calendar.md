# <a name="policy-error-when-sharing-a-calendar"></a>Richtlinienfehler beim Freigeben eines Kalenders

Wenn Sie versuchen, Ihren Kalender freizugeben, und die Fehlermeldung "Richtlinie lässt die Gewährung von Berechtigungen auf dieser Ebene an einen oder mehrere Der Empfänger nicht zu" angezeigt wird, lesen Sie [dieses Thema](https://support.microsoft.com/help/3187524/policy-does-not-allow-granting-permissions-at-this-level-to-one-or-mor).
