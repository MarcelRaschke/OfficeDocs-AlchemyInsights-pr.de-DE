# <a name="public-folder-access-from-mobile-devices"></a>Zugriff auf öffentliche Ordner von mobilen Geräten

Der Zugriff auf öffentliche Ordner funktioniert nicht über mobile Geräte. Verwenden Sie OWA (Outlook im Web) oder Outlook für Windows Desktop- oder Outlook für Mac-Clients, um auf öffentliche Ordner zuzugreifen.

Wenn Sie ähnliche Funktionen wie öffentliche Ordner wünschen und inhalte auf mobilen Geräten zugänglich sein möchten, finden Sie unter ["Informationen zu Microsoft 365 Gruppen](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2)" eine bessere Alternative.
