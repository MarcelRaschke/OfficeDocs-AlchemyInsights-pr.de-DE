# <a name="public-folder-removed-from-outlook-on-the-web-owa-favorites"></a>Öffentlicher Ordner, der aus den Outlook im Web-Favoriten (OWA) entfernt wurde

Es gibt ein bekanntes Problem, bei dem öffentliche Ordner bei jeder Abmelde- oder Bildschirmaktualisierung aus den Favoriten in OWA entfernt werden. Das Problem wird in Kürze behoben. Bis dahin fügen Sie den Ordner für den Zugriff aus OWA erneut hinzu.
