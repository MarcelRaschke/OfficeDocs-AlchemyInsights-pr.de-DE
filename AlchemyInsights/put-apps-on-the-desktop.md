# <a name="put-apps-on-the-desktop"></a>Setzen von Apps auf dem Desktop

Suchen Sie im  Startmenü nach der App, und greifen Sie das App-Symbol an, indem Sie die linke Maustaste gedrückt halten und das Symbol auf den Desktop verschieben.

Sie können auch die **Windows-TASTE + R** drücken und dann den folgenden Befehl eingeben:

`shell:appsfolder`

Dadurch wird der Ordner **"Anwendungen"** im **Datei-Explorer geöffnet.** Sie können nun das App-Symbol per Drag and Drop auf den Desktop ziehen und ablegen.
