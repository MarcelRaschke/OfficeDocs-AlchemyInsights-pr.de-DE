# <a name="receive-copy-of-your-billing-statement-in-email"></a>Empfangen einer Kopie Ihrer Abrechnung per E-Mail

Sie können auf Wunsch eine Kopie Ihrer Rechnung als E-Mail-Anlage erhalten. In diesem Fall sollten Sie Folgendes beachten:
  
- Es wird eine E-Mail sowohl an die primäre als auch die alternative E-Mail-Adresse jeder Person gesendet, die als Benachrichtigungsempfänger aufgeführt ist.

- Sie erhalten zwei separate E-Mails: eine, mit der Sie wissen, dass Ihre Rechnung bereit ist, mit Anweisungen zur Anmeldung beim Admin Center, um sie zu sehen, und eine zweite E-Mail, die die Abrechnungsabrechnung als Anlage enthält.

So empfangen Sie Ihre Abrechnung per E-Mail:
  
1. Navigieren Sie im Admin Center zur Seite **Abrechnung** \> [Abrechnungsbenachrichtigungen](https://go.microsoft.com/fwlink/p/?linkid=853212).

2. Wechseln **Sie unter Abrechnungsabrechnung als E-Mail-Anlage empfangen?** zu **Ein**. Sie können den Empfang von Rechnungs-E-Mails jederzeit beenden, indem Sie die Umschaltfläche auf **Aus** setzen.
