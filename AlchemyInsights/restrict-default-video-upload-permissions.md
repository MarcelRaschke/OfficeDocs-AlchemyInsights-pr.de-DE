# <a name="restrict-default-video-upload-permissions"></a>Einschränken der Standardmäßigen Videouploadberechtigungen

Wenn Benutzer ein Video über das Stream-Portal außerhalb des Kontexts eines Gruppen- oder Gruppenkanals in Microsoft Stream hochladen, ist das Kontrollkästchen "Jedem in Ihrem Unternehmen das Anzeigen dieses Videos erlauben" standardmäßig aktiviert. Dadurch kann das Video von allen in Ihrem Unternehmen angezeigt werden.

- [Festlegen der Standardmäßigen Videouploadberechtigungen](https://docs.microsoft.com/stream/default-video-permissions)
