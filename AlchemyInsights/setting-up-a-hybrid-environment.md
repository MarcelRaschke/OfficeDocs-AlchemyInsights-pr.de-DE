# <a name="setting-up-a-hybrid-environment"></a>Einrichten einer Hybridumgebung

Damit Ihre Benutzer nahtlos zusammenarbeiten können, unabhängig davon, ob ihre Daten auf lokalen Servern oder in Microsoft 365 gespeichert sind, müssen Sie möglicherweise eine Hybridumgebung einrichten. Um einen Überblick über Ihre Optionen zu erhalten, schauen Sie sich [Microsoft 365 Integration in lokale Umgebungen an](https://docs.microsoft.com/office365/enterprise/office-365-integration).
