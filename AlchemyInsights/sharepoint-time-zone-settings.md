# <a name="sharepoint-time-zone-settings"></a>SharePoint-Zeitzoneneinstellungen

Als Websitebesitzer oder Websitesammlungsadministrator können Sie SharePoint-Zeitzonen (regionale Einstellungen) für alle Benutzer der Website auswählen. Einzelne Benutzer können die Standardeinstellungen der Website verwenden oder eigene persönliche Einstellungen angeben. 

- **Ändern der Zeitzone auf Websiteebene**: Weitere Informationen finden Sie [unter Ändern der Zeitzone oder anderer regionaler Einstellungen](https://support.microsoft.com/office/change-regional-settings-for-a-site-e9e189c7-16e3-45d3-a090-770be6e83c1a). 
- **Ändern der Zeitzone auf Benutzerebene**: Siehe ["Ändern Ihrer persönlichen Sprach- und Regionseinstellungen"](https://support.microsoft.com/office/change-your-personal-language-and-region-settings-caa1fccc-bcdb-42f3-9e5b-45957647ffd7).
