# <a name="adding-your-domain"></a>Hinzufügen Ihrer Domäne

Das Konfigurieren Ihrer Domäne kann ein wenig schwierig sein. Wir können Ihnen helfen.

Wenn Sie dies noch nicht getan haben, empfehlen wir Ihnen, sich an einen unserer Microsoft-Spezialisten zu wenden, um Ihnen zu helfen:

- Konfigurieren/Verbinden Ihrer Domäne mit Microsoft 365

*Weitere Informationen zum Konfigurieren Ihrer Domäne für Microsoft 365:* [Hinzufügen einer Domäne zu Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/setup/add-domain)

Wir sind für Sie da, wenn Sie weitere Unterstützung benötigen. Um sich **mit einem Microsoft-Spezialisten** für weitere Hilfe zu verbinden, wählen Sie einen Chat mit einem Microsoft-Spezialisten aus, und **klicken Sie unten auf "Live-Chat** ".
