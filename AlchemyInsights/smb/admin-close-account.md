# <a name="close-business-account"></a>Geschäftskonto schließen

Wenn Sie Ihr Microsoft-Konto schließen, werden alle Informationen zu Ihrem Konto gelöscht. Diese Informationen umfassen Abonnements, Lizenzen, Zahlungsmethoden, Benutzer und Benutzerdaten.

Es gibt mehrere Schritte, die Sie ausführen müssen, um Ihr Konto vollständig zu schließen:

1. Benutzer löschen.
2. Alle aktive Abonnements kündigen.
3. Alle deaktivierten Abonnements löschen.
4. Die Mehrstufige Authentifizierung deaktivieren.
5. Das Verzeichnis in Azure Active Directory löschen.

Informationen zum schrittweisen Prozess und weitere Informationen zum Schließen und Löschen Ihres Geschäftskontos finden Sie unter ["Schließen Ihres Kontos](https://docs.microsoft.com/microsoft-365/commerce/close-your-account)".

Wir sind für Sie da, wenn Sie weitere Unterstützung benötigen. Um sich **mit einem Microsoft-Spezialisten** für weitere Hilfe zu verbinden, wählen Sie einen Chat mit einem Microsoft-Spezialisten aus, und **klicken Sie unten auf "Live-Chat** ".
