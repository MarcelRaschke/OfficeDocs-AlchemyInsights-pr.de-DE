# <a name="admin-takeover"></a>Administratorübernahme

Wenn Sie ein Administrator sind und einen nicht verwalteten Mandanten übernehmen möchten, der durch eine Self-Service-Benutzeranmeldung erstellt wurde, können Sie dies mit einer internen Administratorübernahme tun.

Wenn Sie dies noch nicht getan haben, empfehlen wir Ihnen, sich an einen unserer Microsoft-Spezialisten zu wenden, um Ihnen zu helfen:

- Durchführen einer internen Domänenübernahme.

*Weitere Informationen zur Domänenübernahme:* [Durchführen einer internen Administratorübernahme](https://docs.microsoft.com/microsoft-365/admin/misc/become-the-admin)

Wir sind für Sie da, wenn Sie weitere Unterstützung benötigen. Um sich **mit einem Microsoft-Spezialisten** für weitere Hilfe zu verbinden, wählen Sie einen Chat mit einem Microsoft-Spezialisten aus, und **klicken Sie unten auf "Live-Chat** ".
