# <a name="create-shared-mailbox"></a>Freigegebenes Postfach erstellen

Freigegebene Postfächer lassen sich auf einfache Weise erstellen, damit eine Gruppe von Personen E-Mails einer gemeinsamen E-Mail-Adresse wie "info@contoso.com" überwachen und senden kann. Wenn eine Person in der Gruppe auf eine Nachricht antwortet, die an das freigegebene Postfach gesendet wurde, wird als Absender der E-Mail das freigegebene Postfach angegeben, nicht der einzelne Benutzer. 

**Erstellen eines freigegebenen Postfachs und Hinzufügen von Mitgliedern**

1. Melden Sie sich mit einem globalen Administrator oder mit einem Exchange-Administratorkonto an. Wenn Die Meldung *"Sie sind nicht berechtigt, auf diese Seite zuzugreifen oder diese Aktion auszuführen"* angezeigt wird, sind Sie kein Administrator.
2. Wechseln Sie im Admin Center zur Seite **Teams &** **GroupsShared-Postfächer** > .
3. Wählen Sie auf der Seite **"Freigegebene Postfächer** " die Option "+ **Freigegebenes Postfach hinzufügen" aus**. Geben Sie einen Namen für das freigegebene Postfach ein. Es wird eine Standard-E-Mail-Adresse bereitgestellt, die Sie bei Bedarf jedoch bearbeiten können.
4. Wählen Sie **Änderungen speichern** aus. Es kann einige Minuten dauern, bis Sie Mitglieder hinzufügen können.
5. Wählen Sie unter **Nächste Schritte** die Option **Mitglieder zu diesem Postfach hinzufügen** aus. Mitglieder sind die Personen, welche eingehende E-Mails für dieses freigegebene Postfach und die ausgehenden Antworten anzeigen können.
6. Wählen Sie die Schaltfläche **+ Mitglieder hinzufügen** aus. Setzen Sie ein Häkchen neben diejenigen Personen, die dieses freigegebene Postfach nutzen sollen, und klicken Sie dann auf Speichern.
7. Wählen Sie **Schließen** aus.

Weitere Informationen zum Erstellen eines freigegebenen Postfachs finden [Sie unter "Erstellen eines freigegebenen Postfachs – Microsoft 365 Administrator](https://docs.microsoft.com/microsoft-365/admin/email/create-a-shared-mailbox)".

Wir sind für Sie da, wenn Sie weitere Unterstützung benötigen. Um sich **mit einem Microsoft-Spezialisten** für weitere Hilfe zu verbinden, wählen Sie einen Chat mit einem Microsoft-Spezialisten aus, und **klicken Sie unten auf "Live-Chat** ".
