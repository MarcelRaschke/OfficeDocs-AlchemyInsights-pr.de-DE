# <a name="modify-number-of-licenses"></a>Anzahl der Lizenzen ändern

Sie müssen entweder ein globaler Administrator oder ein Abrechnungsadministrator sein, um die Aufgaben ausführen zu können.

1. Navigieren Sie im Admin Center zur Seite **Abrechnung** > **Ihre Produkte**.
2. Suchen Sie auf der Registerkarte " **Produkte** " das Abonnement, für das Sie Lizenzen kaufen oder entfernen möchten. Wählen Sie die drei Punkte (weitere Aktionen) und dann **"Lizenzen kaufen**" aus.
3. Wenn Sie die Anzahl der Lizenzen verringern möchten, wählen Sie oben im Bereich **"Lizenzen kaufen** " die Option **"Lizenzen entfernen"** aus.
4. Um Lizenzen zu kaufen oder zu entfernen, geben Sie unter **"Neue Menge** " im Feld " **Lizenzen insgesamt** " die Gesamtanzahl der Lizenzen ein, die Sie für dieses Abonnement benötigen.
5. Klicken Sie auf **Speichern**.

*Weitere Informationen zum Ändern der Anzahl von Lizenzen/Abonnements:* [Kaufen oder Entfernen von Lizenzen](https://docs.microsoft.com/microsoft-365/commerce/licenses/buy-licenses)

Wir sind für Sie da, wenn Sie weitere Unterstützung benötigen. Um sich **mit einem Microsoft-Spezialisten** für weitere Hilfe zu verbinden, wählen Sie einen Chat mit einem Microsoft-Spezialisten aus, und **klicken Sie unten auf "Live-Chat** ".
