# <a name="onedrive-access-management"></a>OneDrive Zugriffsverwaltung

Dateien und Ordner können freigegeben werden, indem jemandem ein Freigabelink gesendet wird, oder indem ihm direkter Zugriff auf die Datei oder den Ordner erteilt wird.

- Wenn Sie der Dateibesitzer sind, können Sie die Freigabe der Datei oder des Ordners beenden.
- Wenn jemand direkten Zugriff erhalten hat oder über einen bestimmten Personenlink Zugriff hat, können Sie auch die Freigabeberechtigungen zwischen Ansicht und Bearbeitung ändern.

*Weitere Informationen zum Freigeben Ihrer Dateien in OneDrive:*
[Freigeben von OneDrive oder SharePoint Dateien oder Ordnern oder Ändern von Berechtigungen](https://support.microsoft.com/office/stop-sharing-onedrive-or-sharepoint-files-or-folders-or-change-permissions-0a36470f-d7fe-40a0-bd74-0ac6c1e13323)

Wir sind für Sie da, wenn Sie weitere Unterstützung benötigen. Um sich **mit einem Microsoft-Spezialisten** für weitere Hilfe zu verbinden, wählen Sie einen Chat mit einem Microsoft-Spezialisten aus, und **klicken Sie unten auf "Live-Chat** ".
