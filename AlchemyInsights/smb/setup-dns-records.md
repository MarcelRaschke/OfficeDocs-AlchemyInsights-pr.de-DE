# <a name="setting-up-dns-records"></a>Einrichten von DNS-Einträgen

Das Einrichten Ihrer DNS-Einträge kann etwas schwierig sein. Wir können Ihnen helfen.

Wenn Sie dies noch nicht getan haben, empfehlen wir, sich an einen unserer Microsoft-Spezialisten zu wenden, um Ihnen zu helfen:

- Einrichten von DNS-Einträgen, z. B. MX-, TXT-, A-, SPF-Einträge
- Ändern Sie Nameserver, um Microsoft 365 einzurichten.

*Erfahren Sie mehr über das Einrichten Ihrer DNS-Einträge für Microsoft 365:*

- [Hinzufügen von DNS-Einträgen zum Verbinden Ihrer Domäne](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/create-dns-records-at-any-dns-hosting-provider)
- [Ändern des Namenservers zum Einrichten von Microsoft 365 mit einer beliebigen Domänenregistrierungsstelle.](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/change-nameservers-at-any-domain-registrar)

Wir sind für Sie da, wenn Sie weitere Unterstützung benötigen. Wenn Sie **sich mit einem Microsoft-Spezialisten für weitere Hilfe in Verbindung setzen** möchten, wählen Sie "Chat mit einem Microsoft-Spezialisten" aus, und **klicken Sie dann unten auf "Live-Chat** ".
