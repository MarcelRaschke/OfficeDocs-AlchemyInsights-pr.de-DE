# <a name="setting-up-dns-records"></a>Einrichten von DNS-Einträgen

Das Einrichten Ihrer DNS-Einträge kann ein wenig schwierig sein. Wir können Ihnen helfen.

Wenn Sie dies noch nicht getan haben, empfehlen wir Ihnen, sich an einen unserer Microsoft-Spezialisten zu wenden, um Ihnen zu helfen:

- Einrichten von DNS-Einträgen, z. B. MX-, TXT-, A-, SPF-Einträgen
- Ändern Sie Namenserver, um Microsoft 365 einzurichten.

*Weitere Informationen zum Einrichten Ihrer DNS-Einträge für Microsoft 365:*

- [Hinzufügen von DNS-Einträgen zum Verbinden Ihrer Domäne](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/create-dns-records-at-any-dns-hosting-provider)
- [Ändern des Namenservers zum Einrichten von Microsoft 365 mit einer beliebigen Domänenregistrierungsstelle.](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/change-nameservers-at-any-domain-registrar)

Wir sind für Sie da, wenn Sie weitere Unterstützung benötigen. Um sich **mit einem Microsoft-Spezialisten** für weitere Hilfe zu verbinden, wählen Sie einen Chat mit einem Microsoft-Spezialisten aus, und **klicken Sie unten auf "Live-Chat** ".
