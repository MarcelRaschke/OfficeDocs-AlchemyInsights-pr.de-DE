# <a name="sharepoint-apps"></a>SharePoint-Apps

Als SharePoint oder globaler Administrator in Microsoft 365 können Sie Lösungen aus der SharePoint Store erwerben oder benutzerdefinierte Apps verteilen, die über SharePoint, Microsoft Teams und Viva Connections hinweg verwendet werden können. Der erste Schritt beim Abrufen von Lösungen besteht darin, über einen App-Katalog zu verfügen, den Sie zum Speichern und Verteilen von Lösungen verwenden können.

*Weitere Informationen zu SharePoint Apps:*
[Verwalten von Apps mithilfe des App-Katalogs](https://docs.microsoft.com/sharepoint/use-app-catalog)

Wir sind für Sie da, wenn Sie weitere Unterstützung benötigen. Wenn Sie **sich mit einem Microsoft-Spezialisten für weitere Hilfe in Verbindung setzen** möchten, wählen Sie "Chat mit einem Microsoft-Spezialisten" aus, und **klicken Sie dann unten auf "Live-Chat** ".
