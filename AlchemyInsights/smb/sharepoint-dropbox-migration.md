# <a name="migrate-dropbox-files"></a>Migrieren von Ablagedateien

Wenn Sie die Migration von Dateien aus DropBox noch nicht abgeschlossen haben, empfehlen wir Ihnen, sich an einen unserer Microsoft-Spezialisten zu wenden, um Ihnen dabei zu helfen:

- Migrieren Ihrer Dateien von Dropbox zu Microsoft 365 Business

Weitere Informationen zum [Migrieren von Dropbox mithilfe des Migrations-Managers](https://docs.microsoft.com/sharepointmigration/mm-dropbox-overview)

Wir sind für Sie da, wenn Sie weitere Unterstützung benötigen. Um sich **mit einem Microsoft-Spezialisten** für weitere Hilfe zu verbinden, wählen Sie einen Chat mit einem Microsoft-Spezialisten aus, und **klicken Sie unten auf "Live-Chat** ".
