# <a name="transferring-your-domain"></a>Übertragen Ihrer Domäne

Die Übertragung Ihrer Domäne kann ein wenig schwierig sein. Wir können Ihnen helfen.

Wenn Sie dies noch nicht getan haben, empfehlen wir Ihnen, sich an einen unserer Microsoft-Spezialisten zu wenden, um Ihnen zu helfen:

- Übertragen Sie Ihre Domäne von Microsoft 365 an einen anderen Hostanbieter.

*Weitere Informationen zum Konfigurieren Ihrer Domäne für Microsoft 365:* [Übertragen einer Domäne von Microsoft auf einen anderen Host](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/transfer-a-domain-from-microsoft-to-another-host)

Wir sind für Sie da, wenn Sie weitere Unterstützung benötigen. Um sich **mit einem Microsoft-Spezialisten** für weitere Hilfe zu verbinden, wählen Sie einen Chat mit einem Microsoft-Spezialisten aus, und **klicken Sie unten auf "Live-Chat** ".
