# <a name="solutions-for-error-30180-28"></a>Lösungen für Fehler 30180-28

Möglicherweise wird fehler [30180-28](https://support.office.com/article/47ae453b-677c-412f-9a21-6766555ff4de?wt.mc_id=Alchemy_ClientDIA) während eines Office Updates angezeigt, wenn Sie eine schlechte Internetverbindung haben. Here are some things you can try to fix the problem.
  
- Starten Sie Ihren Computer neu, und [aktualisieren Sie](https://support.office.com/article/2ab296f3-7f03-43a2-8e50-46de917611c5?wt.mc_id=Alchemy_ClientDIA)dann Office .

- Führen Sie die obigen Schritte aus, verwenden Sie jedoch eine kabelgebundene Verbindung, während Sie Office aktualisieren.
