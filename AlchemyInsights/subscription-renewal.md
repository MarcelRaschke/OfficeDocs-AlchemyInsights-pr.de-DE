# <a name="subscription-recurring-billing"></a>Wiederkehrende Abrechnung des Abonnements

Wenn Sie wissen möchten, wann Ihr monatliches oder jährliches Abonnement in Rechnung gestellt wird (wenn **die wiederkehrende Abrechnung** aktiviert ist) oder wann es abläuft (wenn **die wiederkehrende Abrechnung** deaktiviert ist), führen Sie die folgenden Schritte aus:
  
1. Navigieren Sie im Admin Center zur Seite **Abrechnung** \> [Ihre Produkte](https://go.microsoft.com/fwlink/p/?linkid=842054).
2. Suchen Sie auf der Registerkarte " **Produkte** " das Abonnement, über das Sie mehr wissen möchten. In der Spalte **"Abonnementstatus"** wird das Verlängerungs- oder Ablaufdatum angezeigt.
3. Um eine Änderung vorzunehmen, wählen Sie die drei Punkte (weitere Aktionen) und dann die **Option "Wiederkehrende Abrechnung bearbeiten**" aus.
4. Wählen Sie im Bereich " **Wiederkehrende Abrechnung bearbeiten** " **die Option "Ein**", **"Ein", "Einmal verlängern**" oder " **Aus" aus**.
5. Klicken Sie auf **Speichern**.

Weitere Informationen hierzu finden Sie unter [Aktivieren oder Deaktivieren der wiederholten Abrechnung](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/renew-your-subscription).
