# <a name="troubleshooting-outlook-mobile-setup-for-a-shared-mailbox"></a>Problembehandlung Outlook mobilen Setups für ein freigegebenes Postfach

Über die auf dem mobilen Gerät installierte Outlook-App können Sie auf das freigegebene Postfach nicht zugreifen. Möchten Sie dieses Feature verwenden? Teilen Sie dies dem Outlook Engineering-Team mit. [Stimmen Sie hier ab!](https://go.microsoft.com/fwlink/?linked=862116)
  
Sie können ein freigegebenes Postfach nur auf einem mobilen Gerät verwenden, indem Sie Ihren Browser öffnen, sich anmelden und dann zu Outlook im Web wechseln. Anweisungen finden Sie unter [Hinzufügen eines freigegebenen Postfachs in Outlook im Web.](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207) Von Outlook im Web aus können Sie auf das freigegebene Postfach zugreifen.
  
Weitere Informationen: [Erhalten von In-App-Hilfe für Outlook für iOS und Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
