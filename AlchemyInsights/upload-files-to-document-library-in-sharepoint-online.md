# <a name="upload-files-to-onedrive-or-sharepoint"></a>Hochladen Von Dateien zu OneDrive oder SharePoint

Sie können Dateien in SharePoint Online in eine Dokumentbibliothek hochladen, indem Sie sie einfach von Ihrem Computer ziehen und in die Dokumentbibliothek ablegen. Mit OneDrive können Sie Dateien zwischen Ihrem Computer und der Cloud synchronisieren, damit Sie von überall zu Ihren Dateien gelangen können. Sie können ihre synchronisierten Dateien auch direkt im Datei-Explorer verwenden und auf Ihre Dateien zugreifen, auch wenn Sie offline sind. Weitere Informationen zum Hochladen und Synchronisieren finden Sie unter den links unten.
