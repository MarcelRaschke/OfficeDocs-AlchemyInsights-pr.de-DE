# <a name="upload-files-to-onedrive-or-sharepoint"></a>Hochladen von Dateien auf OneDrive oder SharePoint

Sie können Dateien in eine Dokumentbibliothek in SharePoint Online hochladen, indem Sie sie einfach von Ihrem Computer ziehen und in die Dokumentbibliothek ablegen. Mit OneDrive können Sie Dateien zwischen Ihrem Computer und der Cloud synchronisieren, sodass Sie von praktisch überall aus auf Ihre Dateien zugreifen können. Sie können auch direkt in Explorer mit Ihren synchronisierten Dateien arbeiten und auch dann auf Ihre Dateien zugreifen, wenn Sie offline sind. Weitere Informationen zum Hochladen und Synchronisieren finden Sie unter den folgenden Links.
