# <a name="upload-process-overview-in-microsoft-stream"></a>Übersicht über den Uploadprozess in Microsoft Stream

Wenn Sie ein Video hochladen, Microsoft Stream es automatisch in mehrere Bitraten codieren, damit das Video mit adaptivem Bitratenstreaming gestreamt werden kann. Microsoft Stream optimiert die Bitrate und Qualität des wiedergegebenen Videos basierend auf der verfügbaren Bandbreite des Betrachters.

- [Übersicht über den Uploadprozess in Microsoft Stream](https://docs.microsoft.com/stream/upload-process-overview)
