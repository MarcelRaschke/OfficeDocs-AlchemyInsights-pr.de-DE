# <a name="use-microsoft-edge-extensions"></a>Verwenden von Microsoft Edge-Erweiterungen

[Erweiterungen](https://go.microsoft.com/fwlink/?linkid=2135619) sind Add-Ons, die Sie in Microsoft Edge installieren können, um Anzeigen zu blockieren, Kennwörter zu verwalten, Text zu übersetzen und Ihre Browsererfahrung auf viele andere Arten anzupassen.
