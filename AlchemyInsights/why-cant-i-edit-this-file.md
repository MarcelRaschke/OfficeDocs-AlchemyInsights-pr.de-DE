# <a name="why-cant-i-edit-this-file"></a>Warum kann ich diese Datei nicht bearbeiten?

Wenn ein Dokument, ein Arbeitsblatt oder eine Präsentation zur Bearbeitung gesperrt ist und in den Windows-Desktopanwendungen nicht bearbeitet werden kann, wird die Datei möglicherweise von einer anderen Person bearbeitet, oder die Datei ist ausgecheckt.

Werfen Sie einen kurzen Blick auf einige der Gründe [warum Sie die Datei möglicherweise nicht bearbeiten können](https://support.office.com/article/why-can-t-i-edit-this-file-97315f48-aa5e-49d3-a4ae-a14b73daf87b).
